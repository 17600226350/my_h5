# 我的个人网站

这是一个可以直接用 VS Code 打开的静态个人网站项目。

## 怎么打开

1. 用 VS Code 打开这个文件夹：`/Users/ljq/Desktop/android/android/web/mine`
2. 双击打开 `index.html`
3. 如果你安装了 VS Code 插件 Live Server，可以右键 `index.html`，选择 `Open with Live Server`

## 常改的位置

- `index.html`：修改名字、简介、作品、联系方式
- `styles.css`：修改颜色、布局、字体大小
- `script.js`：修改页面交互

## 下一步建议

- 把“你的名字”换成真实名字或昵称
- 把邮箱 `hello@example.com` 换成你的联系方式
- 在“我的作品”里加入真实项目链接
